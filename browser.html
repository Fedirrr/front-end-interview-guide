DOM: Tree representation of HTML.

  • API for manipulation

–©–æ —Ç–∞–∫–µ localStorage?

localStorage: Key-value, persistent.

  • 5MB limit

–©–æ —Ç–∞–∫–µ sessionStorage?

  • Same API as local

local: Permanent across sessions.

–©–æ —Ç–∞–∫–µ cookies?

–©–æ —Ç–∞–∫–µ CORS?

CORS: Cross-Origin Resource Sharing.

  • Headers: Access-Control-Allow-Origin

–©–æ —Ç–∞–∫–µ Same-Origin Policy?

SOP: Restrict cross-site requests.

  • Protocol/host/port same

–©–æ —Ç–∞–∫–µ HTTP methods (GET, POST, PUT, DELETE)?

  • 2xx success, 4xx client, 5xx server

–©–æ —Ç–∞–∫–µ WebSocket?

  • ws:// for real-time

–©–æ —Ç–∞–∫–µ Service Worker?

  • Batch changes

–©–æ —Ç–∞–∫–µ Web Vitals (LCP, FID, CLS)?

FID: First input delay (now INP).

–©–æ —Ç–∞–∫–µ Web Workers?
